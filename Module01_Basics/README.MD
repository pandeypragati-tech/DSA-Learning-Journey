
# Module 01 - Basics

## Overview

This module covers the fundamental programming concepts required for Data Structures and Algorithms. The focus is on problem-solving using conditions, loops, functions, and basic mathematics.

## Topics Covered

* Conditional Statements
* Loops
* Functions
* Number Theory
* Basic Mathematics
* Problem Solving Fundamentals
